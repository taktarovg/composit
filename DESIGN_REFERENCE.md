# ХИМСНАБ КОМПОЗИТ — Дизайн-референс для сайта

> Документ описывает структуру, дизайн и технические решения для сайта группы компаний "Химснаб Композит",
> созданный на основе анализа сайта sibur.ru и адаптированный под бренд-бук компании.

---

## 1. БРЕНД И ЦВЕТОВАЯ ПАЛИТРА

### 1.1 Логотип
- Название: **ХИМСНАБ КОМПОЗИТ**
- Логотип: 4 пазла (квадрат 2x2), каждый элемент — свой цвет
- Шрифт логотипа: жирный, без засечек, тёмно-серый (#4B4C4C)

### 1.2 Основные цвета бренда (из PDF)

| Роль | Цвет | HEX | Описание |
|------|-------|------|----------|
| Primary | Teal | `#1896A3` | Основной фирменный — заголовки, кнопки, акценты |
| Accent | Coral-Red | `#D25646` | Акцентный — CTA-кнопки, важные элементы |
| Secondary | Olive Green | `#95AB63` | Вторичный — иконки, бейджи, декор |
| Secondary | Gold | `#D0A823` | Вторичный — выделение, премиум-элементы |
| Neutral | Dark Gray | `#4B4C4C` | Тексты, заголовки, footer |

### 3.2 Скругления (Border Radius)
- Карточки: `16px` — `20px` (как у SIBUR — мягкие, современные)
- Кнопки: `30px` — `50px` (pill-кнопки, как "ПОДРОБНЕЕ" на SIBUR)
- Инпуты: `12px`
- Изображения в карточках: `16px`

### 3.3 Тени
```css
/* Карточки */
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.06);

/* Карточки при hover */
box-shadow: 0 8px 24px rgba(24, 150, 163, 0.15);

/* Header при скролле */
box-shadow: 0 2px 16px rgba(0, 0, 0, 0.08);
```

### 3.4 Стиль кнопок
```

            hover: bg: #147b86, transform: translateY(-2px)

            hover: bg: #b84a3d
```

### 3.5 Стиль карточек
- Белый фон, скругление 16-20px
- При hover: лёгкий подъём (translateY(-4px)) + усиление тени
- Для карточек отраслей: фоновое фото + тёмный градиент снизу + белый текст
- Стрелка-кнопка (→) в нижнем левом углу карточки (как у SIBUR)

## 4. АНИМАЦИИ И ИНТЕРАКТИВНОСТЬ

### 4.1 Scroll-анимации
- **Fade-in + slide-up:** Элементы появляются снизу при скролле (offset: 50px, duration: 600ms)
- **Staggered:** Карточки в сетке появляются последовательно (delay: 100ms между элементами)
- **Счётчики:** Анимация числовых значений от 0 до целевого при видимости

### 4.2 Hover-эффекты
- **Карточки:** подъём + усиление тени + лёгкое увеличение фото внутри (scale: 1.05)
- **Кнопки:** смена цвета + лёгкий подъём
- **Ссылки в навигации:** подчёркивание анимированной линией снизу
- **Карточки отраслей:** teal-оверлей усиливается, стрелка сдвигается вправо

### 4.3 Hero-секция
- Возможна лёгкая параллакс-анимация фонового изображения
- Или плавная смена фотографий (crossfade slideshow, 5-7 секунд)

### 4.4 Header
- При скролле: появляется тень, фон становится полностью непрозрачным белым
- Smooth-transition на все изменения

- НЕ чисто белый, а **светло-серый** (`#F7F8F9` или `#f0f1f2`) — как у SIBUR
- Карточки и секции на белом фоне выделяются на этом фоне

### 8.2 Карточки-ссылки (Quick Links)
- 4 штуки в ряд, каждая имеет свой цвет фона
- Цвета: teal (#1896A3), coral (#D25646), белый/светлый, teal
- Содержат: название + стрелку-кнопку (→) в нижнем левом углу
- Скруглённые углы (~16px)

### 8.3 Карточки отраслей
- Фоновое фото на всю карточку
- Полупрозрачный тёмный градиент снизу
- Название отрасли белым шрифтом (bold) в верхнем левом углу
- Стрелка-кнопка (→) в нижнем левом углу
- При hover: zoom фото + усиление оверлея

### 8.4 Новости в сайдбаре
- Справа от hero
- Каждая новость: заголовок (bold) + дата + декоративный круг/элемент
- Кнопка "Все новости" внизу

### 8.5 Баннер R&D
- Полноширинный блок с teal-градиентом
- Фоновое фото справа (с прозрачностью/размытием)
- Текст и кнопка слева
- Кнопка "ПОДРОБНЕЕ" в стиле outline (рамка + текст)
- Pill-форма кнопки (border-radius: 50px)

### 8.6 Footer
- Тёмный фон (#2e2e2e)
- 3 колонки: юр. информация + контакты | ссылки документов | подписка + соцсети
- Соцсети: круглые иконки
- Кнопка "ПОДПИСАТЬСЯ" с кругом-стрелкой

## 10. ПРИМЕЧАНИЯ

1. **Шрифт Inter** — можно подключить через `next/font/google`, он отлично подходит для корпоративных сайтов
2. **Изображения** — на этапе разработки использовать placeholder-изображения (unsplash: polymers, composites, industrial)
3. **SEO** — использовать Next.js Metadata API для title, description, OG-изображений
4. **Производительность** — Image component из Next.js, lazy loading, оптимизация шрифтов
5. **Accessibility** — Semantic HTML, aria-labels, focus-visible стили, contrast ratio >= 4.5:1
